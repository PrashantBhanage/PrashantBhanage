## 💼 Projects:

### 🔐 FinTrack API *(Finance Backend System)*
A secure, role-based REST API backend for a finance dashboard system built with **Java 17, Spring Boot 3, Spring Security, JWT, and MySQL**. Features RBAC with VIEWER, ANALYST, and ADMIN roles, full CRUD for transactions, users, and categories, a dashboard returning income/expenses/monthly trends, and full Swagger UI documentation.

> `Java` `Spring Boot 3` `Spring Security` `JWT` `MySQL` `Swagger UI` `Docker`

🔗 [View Repository](https://github.com/PrashantBhanage/finance-api)

### 🧠 NeuroLearn *(Java Learning Platform)*
A Java-based learning platform designed to deliver structured lessons, quizzes, and subject-wise content through a clean web interface. Built using **MVC architecture, DAO pattern, Servlet backend, and JDBC database integration** with a responsive frontend featuring a toggle-based dark mode and central dashboard for subject and module navigation.

> `Java` `Servlet` `MVC` `DAO Pattern` `JDBC` `HTML` `CSS` `JavaScript`
